# 📊 Determinants-of-Multiple-Sexual-Partnership-among-Undergraduate-Students-(OAU)-

## 🔍 Project Overview:
Analyzed determinants of multiple sexual partnerships using survey data. Utilized Excel for data cleaning, and STATA/SPSS for statistical analysis to identify key socio-demographic and behavioral predictors for public health insights.
__

## 🎯 Objectives
- Determine the prevalence of MSP among students
- Examine the effect of age at sexual debut on MSP
- Assess the influence of marital status, religion, income, and living arrangement
______

## 🛠 Tools Used
- Excel (Data Cleaning)
- STATA (Statistical Analysis)
- SPSS (Supporting Analysis)
______

## 🧹 Data Cleaning
- Removed missing values
- Encoded categorical variables
- Structured dataset for statistical analysis
______

## 📈 Analysis Performed
- Descriptive statistics
- Cross-tabulation
- Pearson Chi-square test
______

## 🔑 Key Findings
- Early sexual debut is significantly associated with higher likelihood of multiple sexual partnerships (p < 0.05)
- Living arrangement has a statistically significant influence on MSP
- Gender, marital status, religion, and income showed no significant association with MSP
